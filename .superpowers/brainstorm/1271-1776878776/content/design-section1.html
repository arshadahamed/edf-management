Tab behavior: "View History" tab = existing read-only table + search/filter/CSV export (no changes). "Manage Payments" tab = accordion CRUD with all operations.
Progress bar: Visual fill shows paid_amount ÷ expected_amount at a glance. Green = full, amber = partial, empty = unpaid.
Month actions: "Delete Month" button shows an inline red confirm strip (replacing the button) — must confirm before deletion. Edit fields appear inline in the two-section layout.
Installment actions: Edit icon converts row to inline input grid. Delete icon shows inline confirm on that row. "+ Add Installment" appends a blank input row at the bottom.
New month form: "+ Add Month" reveals a full-width form above the accordion. Month picker prevents duplicate months.
